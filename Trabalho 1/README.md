# Trabalho 1 - Computação Concorrente

## Descrição:

O código aqui apresentado visa calcular a eliminação gaussiana de uma matriz gerada por números aleatórios. O usuário digita a dimensão e a quantidade de threads que vamos utilizar, o programa gera uma matriz, realiza o escalonamento, faz substituição reversa para encontrar os valores do vetor x e MMQ para verificar se Ax - b = 0, ou seja, se está tudo certo com a resposta.
